O DOM é um acrônimo para Document Object Model - Não funciona dentro do Node JS, só funciona quando Javascript roda no navegador
É possível criar uma árvore DOM que possui a raiz (Window), onde estão:
-Location (url do site)
-Document(documento autal)
-History(grava histórico de navegação)

Sempre que se referencia o ID se usa um '#' e para as CLASSES>, se usa '.'
